# Travel Back

Proyecto base de Spring Boot para el backend.

## Requisitos

- Java 21
- Maven 3.9 o superior

## Ejecutar

```powershell
mvn spring-boot:run
```

La API queda disponible por defecto en:

```text
http://localhost:8080
```

Endpoint de prueba:

```text
GET http://localhost:8080/api/health
```
