
# Requerimientos do sistema – ExploraMas

## 1- Descrición Xeral

ExploraMas é unha aplicación web de viaxes que permite aos usuarios buscar destinos, comparar opcións e planificar viaxes completas nun único lugar. A plataforma estará dispoñible en navegadores web e permitirá acceder tanto a usuarios rexistrados como non rexistrados.

O obxectivo principal é facilitar a planificación de viaxes mediante unha interface sinxela.

## 2- Funcionalidades

| Acción | Descrición |
|-------|------------|
| Rexistro de usuarios | Permite crear unha conta na plataforma |
| Inicio de sesión | Acceso de usuarios rexistrados |
| Busca de destinos | Buscar viaxes segundo preferencias |
| Reserva de viaxes | Posibilidade de reservar servizos |
| Xestión de perfil | Modificar datos do usuario |
| Administración | Xestión da web por parte do administrador |

## 3- Tipos de usuarios

- Usuario anónimo: pode buscar destinos e ver información básica.
- Usuario rexistrado: pode gardar preferencias, recibir recomendacións e facer reservas.
- Administrador: xestiona contido, usuarios e funcionamento da plataforma.

## 4- Contorno operacional

Para utilizar ExploraMas é necesario:

- Navegador web actualizado (Chrome, Firefox, Edge…)
- Conexión a internet
- Dispositivo (ordenador, tablet ou móbil)

Non se require software adicional.

## 5- Normativa

ExploraMas cumprirá coa normativa vixente:

- LOPDGDD (Lei Orgánica de Protección de Datos)
- GDPR (Regulamento Xeral de Protección de Datos da UE)

A web incluirá:
- Aviso legal
- Política de privacidade
- Política de cookies

Garantirase o tratamento seguro dos datos dos usuarios.

## 6- Melloras futuras

- Aplicación móbil
- Sistema de valoracións e opinións
- Integración con máis provedores de viaxes
- Suxestión de viaxes personalizadas
